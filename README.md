## Sobre o Repositório

## Instalação

```bash
$ npm install
```

## Rodar ☸️ o app

```bash
# modo desenvolvimento
$ npm run start

# modo "watch"
$ npm run start:dev

# modo produção
$ npm run start:prod
```

## Testando

```bash
# testes unitários
$ npm run test

# tests e2e
$ npm run test:e2e

# verificar a cobertura de tests
$ npm run test:cov
```
